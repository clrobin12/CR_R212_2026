Rapport — TP3
Points clés
1. Le stockage local
J'ai retenu qu'on peut utiliser le localStorage pour garder des données même après avoir fermé la page. C'est pratique pour éviter que tout s'efface quand on actualise.

2. La transformation des données
Il faut transformer les tableaux en texte avec JSON.stringify pour que le navigateur accepte de les stocker, et faire l'inverse avec JSON.parse pour les récupérer.
